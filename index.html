Importar dados

Selecione o arquivo .html exportado do Google Sheets (ou um .xlsx exportado por este sistema). Os dados serão mesclados — datas já existentes serão ignoradas.

🗑️ Apagar registros por mês/ano

Remove todos os lançamentos de um mês específico da loja ativa. Use antes de reimportar um mês com dados incorretos.

⚠️ Zona de perigo — Reset completo

Apaga todos os lançamentos da loja ativa de uma vez. Use quando os dados foram importados incorretamente e você quer começar do zero.
Esta ação é irreversível.

Para trocar de projeto, edite o objeto FIREBASE_CONFIG no início do JavaScript e atualize ADMIN_EMAIL.
